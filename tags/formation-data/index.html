    Adding Formation Data to a Well Log Plot

    Introduction Taking well log plots one step further using Python and matplotlib Well log plots are a common visualization tool within geoscience and petrophysics. They allow easy visualization of data (for example, Gamma Ray, Neutron Porosity, Bulk Density, etc) that has been acquired along the length (depth) of a wellbore.
